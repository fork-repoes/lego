# 乐高

#### Description
软件开发就像玩乐高。
1. 明确要搭建的作品（明确功能目标）
2. 找到所需要的积木块（开发所需要的组件）
3. 将积木块组装起来（将业务组件连接起来，完成业务流程）

当目标改变时：
1. 将作品还原成 积木块（系统需求发生变更）
2. 寻找新的积木块（开发新的组件）
3. 将新旧积木块组装成作品（将新旧功能组件连接起来，完成新需求）
